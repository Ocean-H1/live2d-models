# Live2D Models

收集一些喜欢的Live2d模型，主要用于个人网站的看板娘, 本仓库fork自https://github.com/oh-my-live2d/live2d-models

## 使用方法

使用`https://cdn.jsdelivr.net/gh/Ocean-H1/live2d-models/models/module_name/xxx.model3.json`访问模型的根配置文件即可
